다범주 로짓 회귀

명목형 - 기준범주로짓모형 (다항로지스틱회귀)
# multinomial regression in R 
library(nnet)
model <- multinom(Y ~ X, data)
summary(model)

# coefficient interpretaion - odds ratio
exp(coef(model))
순서형 - 누적로짓모형 
# ordinal logistic regression in R 
library(MASS)
model <- plor(Y ~ X, data, Hess=TRUE)
summary(model)

# coefficient interpretation - odds ratio
exp(coef(model))

ZIP 회귀(제로팽창포아송 회귀)

0이 너무 많아서 영과잉 문제가 나타난 도수자료에 대해 포아송 회귀모형을 돌릴 때, 제대로 적합하지 않는 경우가 발생할 수 있으므로, 이 경우 ZIP 회귀 모형을 돌려보는 것도 방법!

# ZIP regression in R 
library(pscl)
model <- zeroinfl(y ~ x1 + x2 | z1 + z2 + z3, data) 
# x1 + x2 | z1 + z2 + z3 : count data model y ~ x1 + x2 conditional on (|) the zero-inflation model y ~ z1 + z2 + z3
